This third-party repository provides [XLibre](https://xlibre.org) x86_64 binary  packages for [Manjaro](https://manjaro.org). XLibre is the community-managed display server for the [X Window System Protocol Version 11 (Wikipedia)](https://en.wikipedia.org/wiki/X_Window_System_core_protocol), in short, X11. You can learn more about XLibre at [xlibre.net](https://xlibre.net/).

Please see our homepage on [**how to install and use**](https://xlibre-manjaro.github.io/) the packages.
